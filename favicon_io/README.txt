🔧 COMMENT UTILISER:

OPTION 1: Utiliser directement les SVG (RECOMMANDÉ)
-----------------------------------------------------
Les fichiers SVG sont vectoriels et s'adaptent à toutes les tailles.
Vous pouvez les utiliser directement dans votre HTML:

<link rel="icon" type="image/svg+xml" href="favicon_io/favicon-32x32.svg">

OPTION 2: Convertir en PNG/ICO
-----------------------------------------------------
Si vous avez besoin de formats PNG ou ICO:

1. Aller sur: https://cloudconvert.com/
2. Uploader le fichier SVG
3. Choisir le format de sortie (PNG ou ICO)
4. Télécharger le fichier converti

OPTION 3: Utiliser RealFaviconGenerator (MEILLEUR)
-----------------------------------------------------
Pour générer un package complet avec tous les formats:

1. Aller sur: https://realfavicongenerator.net/
2. Uploader: android-chrome-512x512.svg
3. Personnaliser les options
4. Télécharger le package ZIP complet
5. Copier le code HTML fourni

🔄 CONVERSION SVG → PNG:
═══════════════════════════════════════════════════════════════

Méthode 1: CloudConvert (en ligne)
https://cloudconvert.com/svg-to-png

Méthode 2: Inkscape (logiciel gratuit)
1. Télécharger Inkscape: https://inkscape.org/
2. Ouvrir le fichier SVG
3. File → Export PNG Image
4. Choisir la taille
5. Exporter

https://www.photopea.com/
